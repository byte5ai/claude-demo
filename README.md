# Claude & Claude Code — Tech Talk 2026

Slide-Deck für den byte5 Tech Talk **„Claude & Claude Code — vom Chat
zum Agenten-Stack"** von Marcel Wege.

**Live:** https://byte5ai.github.io/claude-demo/ · PDF unter `/deck.pdf`

## Worum geht's?

Ein 60-Minuten-Streifzug durch das Claude-Ökosystem für Devs — vom
Modell bis zum getunten Agenten. Roter Faden in acht Phasen:

1. **Was ist Claude?** — Modellfamilie, SWE-Bench-Einordnung, warum
   Benchmarks nur die halbe Wahrheit sind.
2. **Touchpoints** — vom Chat zum Agenten: wo begegnet einem Claude
   überhaupt?
3. **Cowork** — Claude Desktop App im Arbeits-Modus, aktiviert per
   `/setup-cowork` mit rollenbasierten Plugin-Bundles (Beispiel:
   Operations).
4. **Claude Code — warum tunen?** — gleicher Preis, andere Liga. Drei
   Hebel: Tools, Workflows, Kontext.
5. **Der MCP-Stack** — der größte Hebel. Code-Graph-MCPs als
   Token-Sparer, Memory-MCPs für Persistenz zwischen Sessions, der
   pragmatische Katalog.
6. **Skills · Hooks · Subagents** — wiederverwendbare Workflows, mehr
   Hände, Automatisierung um Claude herum.
7. **Spec-Driven Development** — Vibe-Coding skaliert nicht. Live-Demo
   mit GitHub Spec-Kit:
   `/speckit.constitution → .specify → .plan → .tasks → .implement`
   baut eine streaming Next.js Chat-App.
8. **Erfahrungsaustausch** — Diskussions-Trigger statt Schluss-Slide.
